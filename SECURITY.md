# 보안 정책 (Security Policy)

Quetzalcoatl은 **모델 중립 프롬프트 스킬**(마크다운)이며 실행 코드·네트워크 종단·비밀값을 포함하지 않는다. 그렇지만 "AI가 비가역·외부 작업을 사람 승인 없이 하도록 유도되는" 문제는 이 스킬의 핵심 보안 사안으로 다룬다.

## 보고 대상 (security-relevant)

- 스킬 지시가 §1.6 **안전 게이트를 우회·약화**시킬 수 있는 문구.
- 자율 실행(§18)이 사용자 승인 없이 **비가역·외부 작업**을 하도록 유도될 수 있는 경로.
- **프롬프트 인젝션**으로 게이트·교차검증을 무력화하는 시나리오.
- 매니페스트(`plugin.json`/`marketplace.json`)나 설치 절차의 무결성 문제.

## 보고 방법

- 공개 이슈로 올리기 전에 저장소 소유자 [@chquandogong](https://github.com/chquandogong)에게 **비공개로** 알리는 것을 권장한다.
- GitHub **Security 탭 → Private vulnerability reporting**을 사용할 수 있다.
- 재현 경로(어떤 입력이 어떤 게이트를 어떻게 우회하는지)를 포함해 주면 좋다.

## 지원 버전

최신 마이너 릴리스(`v1.3.x`)를 지원한다. 과거 버전은 최신으로 업데이트를 권장한다(`docs/05-ops/RUNBOOK.md`).

## 범위 밖

- 사용자가 자신의 환경에서 **자율 수준을 임의로 높여** 발생한 결과. 자율 수준·허용목록은 강제되는 락이 아니라 약속이며, 최종 안전장치는 §1.6 게이트·커밋 이력·사람의 검토다(SKILL §18.6, §15).
